FAQ

Preguntas frecuentes

¿La página se verá bien en celular?

Sí. El diseño se trabaja primero pensando en una experiencia clara en celular, tablet y computadora.

¿Puede conectarse a WhatsApp?

Sí. Los botones y formularios pueden abrir WhatsApp con un mensaje listo para enviar.

¿Sirve para un negocio pequeño?

Sí. Una web clara ayuda a que un negocio local se vea más confiable y reciba solicitudes con menos fricción.

¿Se puede adaptar a mi marca?

Sí. Se puede adaptar a barberías, tiendas, salones, marcas personales, servicios profesionales y más.

¿Cuánto tarda una página?

Depende del alcance, pero una landing o web de negocio suele tomar entre 7 y 14 días después de definir el contenido.

¿Qué necesito para empezar?

Una idea clara del negocio, servicios, fotos si tienes, datos de contacto y referencias visuales si quieres un estilo específico.

¿Puedo pedir un servicio personalizado?

Sí. Si necesitas una web con estructura, secciones o funciones especiales, se revisa el alcance y se prepara una cotización a medida.

¿Ofreces mantenimiento después de entregar la página?

Sí. Hay planes mensuales para revisión, backups, actualizaciones básicas, cambios pequeños, soporte por WhatsApp y hosting básico opcional.

¿El dominio y los correos están incluidos?

No por defecto. Dominio, correos corporativos, licencias y servicios externos se cotizan aparte para que el alcance quede claro desde el inicio.
